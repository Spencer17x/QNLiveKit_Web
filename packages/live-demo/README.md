# live-demo

互动直播标准接入示例demo

## 目录介绍

* [src/api](./src/api): 业务请求接口
* [src/config](./src/config): 通用配置文件
* [src/layouts](./src/layouts): 页面布局组件
* [src/pages](./src/pages): 页面组件
* [src/router](./src/router): 路由
* [src/styles](./src/styles): 样式
* [src/utils](./src/utils): 工具类
* [App.tsx](./src/App.tsx): 根组件

## 如何运行

```shell
$ pnpm install 
$ pnpm run dev
```
